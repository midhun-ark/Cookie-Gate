Test Website

This page is used to test the ComplyArk consent loader.

Testing Instructions:

  1. Create a website in the ComplyArk Tenant Platform
  2. Add a notice with English (required) and other language translations
  3. Add at least one purpose with translations
  4. Activate the website
  5. Copy the website ID and replace WEBSITE_ID_HERE in this file
  6. Open this page in a browser to see the consent banner

Multi-Language Features:

  • Banner text loads in user's preferred language
  • Falls back to English if translation is missing
  • Language selection persists across sessions
  • Settings panel includes language switcher
